# Slides!
These are the slides I put together for the 2025 class!

1. [installation_and_file_system.pdf](installation_and_file_system.pdf) $-$ installing packages and introducing the UNIX/Linux file system
2. [intro_to_the_shell.pdf](intro_to_the_shell.pdf) $-$ introducing the command line and the shell, and basic bash scripting
3. [handling_data.pdf](handling_data.pdf) $-$ using ASCII files for data and some commands (awk, grep, sed) that help you manipultae them
4. [intro_to_gmt.pdf](intro_to_gmt.pdf) $-$ introduction to the Generic Mapping Tools (GMT) and using it to plot basic maps
5. [gmt_graphs_and_histograms.pdf](gmt_graphs_and_histograms.pdf) $-$ using GMT to plot graphs and histograms
6. [gmt_raster_graphics.pdf](gmt_raster_graphics.pdf) $-$ using GMT to plot raster graphics and topography
7. [intro_to_LaTeX.pdf](intro_to_LaTeX.pdf) $-$ introducing the LaTeX typesetting language and using it to make nicely formatted documents
8. [Ax_equals_b.pdf](Ax_equals_b.pdf) $-$ the basics of linear inverse modeling, least squares and matrix inversion
9. [nonlinear_inverse_modeling.pdf](nonlinear_inverse_modeling.pdf) $-$ solving nonlinear inverse problems with grid searches and optimization
10. [mcmc.pdf](mcmc.pdf) $-$ solving nonlinear inverse problems using Markov Chain Monte Carlo
11. [git.pdf](git.pdf) $-$ versioning of codes and basic use of git
